# PBG Viewer

PBG Viewer is a full-stack MERN application built to showcase modern frontend UI development and backend integration skills. The project focuses on creating a clean, interactive, and responsive web experience while implementing a complete authentication workflow and scalable application structure.

The current version includes a web-based document workspace interface along with a user authentication system that allows users to create accounts and securely sign in. The project has been designed with future expansion in mind, making it easy to integrate additional document management and AI-powered capabilities.

## Purpose

The primary goal of this project is to demonstrate practical development skills across both frontend and backend technologies, including:

- Building responsive and visually structured user interfaces
- Managing application state in React
- Creating reusable component architecture
- Integrating frontend and backend services
- Implementing user authentication workflows
- Working with REST APIs and data exchange
- Structuring a scalable MERN application

## Current Features

- User registration and login system
- Authentication flow with backend integration
- Persistent user session handling
- Responsive and modern user interface
- Web-based document editor workspace UI
- React Context based state sharing
- Frontend and backend integration using API requests

## Planned Features

The project is actively evolving and additional functionality will continue to be added in future updates.

Planned additions include:

- PDF to image conversion
- Image to PDF operations
- Additional document manipulation tools
- AI-assisted document editing
- AI-powered content manipulation and enhancement
- Improved document management workflow
- Expanded user features and productivity tools
